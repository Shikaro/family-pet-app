# Family Pet App

Прототип приложения для родителей и детей 2–10 лет, где ребенок выполняет задания, получает внутриигровую валюту и растит виртуального питомца.

## Идея

- Задачи для ребенка с простыми действиями и полезными привычками
- Родитель контролирует прогресс и видит завершение
- За выполнение даются монетки и уровень питомца
- Монетки можно обменять на реальные призы
- Питомец растет вместе с ребенком и мотивирует к выполнению заданий

## Архитектура

- `backend/` — сервис на Node.js + Express с API
- `frontend/` — фронтенд на React + Vite
- `README.md` — общая документация и инструкции

## Структура

- `backend/src/data.ts` — начальные данные, задания и логика валюты
- `backend/src/routes.ts` — API для задач, наград и питомца
- `frontend/src/App.tsx` — основной пользовательский интерфейс
- `frontend/src/components/` — карточки дашборда, задач, магазина, питомца, истории, режима и челленджа
- `frontend/src/api.ts` — взаимодействие с API
- `backend/src/storage.ts` — файловое хранилище состояния
- `bot/` — Telegram-бот для управления заданиями и наградами

## Возможности

- Режимы «Родитель» и «Ребенок"
- Ежедневный челлендж для семьи
- История действий и событий
- Сохранение состояния на диск
- Telegram-бот для быстрых команд

## Запуск

1. Перейти в папку проекта:
   ```bash
   cd /home/client/projects/Apps
   ```
2. Установить зависимости:
   ```bash
   npm install
   npm install --workspace backend
   npm install --workspace frontend
   ```
3. Запустить бекенд и фронтенд:
   ```bash
   npm run dev --workspace backend
   npm run dev --workspace frontend
   ```

4. Открыть приложение:
   - Фронтенд: `http://localhost:5173`
   - Бэкенд: `http://localhost:4000/api/dashboard`
